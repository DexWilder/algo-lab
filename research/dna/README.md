# Strategy DNA Clustering

*Placeholder for future work.*

## Concept

Cluster strategies by behavioral "DNA" — not by family label, but by actual performance characteristics across market regimes, correlation structure, and trade timing patterns.

## Planned Features

- **Behavioral fingerprint:** PF by regime, avg trade duration, entry time distribution, win rate by session
- **Clustering:** Hierarchical clustering of strategy fingerprints to find true diversifiers
- **Redundancy detection:** Identify strategies that behave identically despite different logic
- **Gap analysis:** Find behavioral profiles missing from the portfolio

## Dependencies

- Requires Phase 8 regime engine (completed)
- Requires ≥5 validated strategies for meaningful clustering
- Regime performance profiles as input features
